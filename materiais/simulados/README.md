# Simulados

Esta pasta reúne simulados e modelos de avaliação que podem ser mantidos publicamente no repositório.

Os arquivos `simulado-*.md` desta pasta são incluídos no índice público do site.

Eles são gerados como documentos A4, não como slides.

Para provas reais ou materiais que não devem ser publicados no GitHub, use a pasta local `privado/avaliacoes/`, que é ignorada pelo Git.
